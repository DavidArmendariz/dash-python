# Environment

First, install `pip` in Linux:

```bash
sudo apt-get install python3-pip
```

## Creating a virtual environment

```bash
python3 -m venv venv
```

Entering to the environment:

```bash
source venv/bin/activate
```

## Installing requirements

```bash
pip install -r requirements.txt
```

## Run the app

```bash
python app.py
```
